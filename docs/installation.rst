Installation

Python Package
==============

Install package with:
  
.. code-block:: bash

   pip install .

or in editable mode:

.. code-block:: bash

   pip install -e .

or with url:

.. code-block:: bash

    pip install git+https://github.com/optimizacion-2-2021-1-gh-classroom/example-python-package-and-sphinx.git
